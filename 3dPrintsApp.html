About the Project

GitHub Video Figma

The 3D Prints App is an intuitive and dynamic mobile application built with React Native. It is designed to help users effortlessly manage their 3D model collections. With powerful features such as dynamic model management, robust image handling, and real-time updates, this app offers a seamless user experience for 3D printing enthusiasts.

Key Features:

  • Add Models: Easily add new models by entering a title, description, and image URLs. The app validates all image URLs, ensuring error-free submissions and allowing multiple images for a detailed representation.
  • Dynamic Model Display: Models are displayed dynamically, reflecting real-time user input without any hardcoded data.
  • Search Functionality: Quickly find models using the dynamic search bar, which filters results based on your input. A handy counter displays the total number of visible models.
  • Details Screen: View each model's details, including a full description and the ability to cycle through multiple images using a dedicated button for an immersive experience.
  • Edit Models: Update existing models with ease. Modify titles, descriptions, and image collections dynamically, ensuring your collection stays up to date.
  • Delete Models: Keep your collection clean and organized with the delete feature, allowing you to remove models effortlessly.

The 3D Prints App is a complete solution for managing your 3D model collection. With its dynamic capabilities and user-friendly design, it empowers users to maintain a well-organized and visually appealing collection. Whether you're adding, editing, or searching for models, this app ensures an efficient and enjoyable experience.
