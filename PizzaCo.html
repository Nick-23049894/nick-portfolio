About the Project

Website GitHub

Nick's Pizza Co. is a simple yet dynamic web application built with React to showcase a pizza shop's menu. It features real-time updates and a user-friendly interface to enhance the customer experience.

Key Features:

  • Search Functionality: Implements useState to filter and display menu items dynamically based on user input, ensuring a seamless search experience.
  • Real-Time Shop Status: Automatically updates the shop's status (open/closed) based on the user's local time, providing accurate information about availability.

Nick's Pizza Co. demonstrates the power of React hooks like useState for state management and offers a practical example of integrating real-time functionality in web applications.
