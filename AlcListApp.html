About the Project

GitHub Video

The React Native Alcohol List App is a dynamic mobile application designed to showcase a categorized collection of popular alcoholic beverages. With a sleek user interface and responsive features, the app allows users to explore different categories of drinks like Whiskey, Vodka, Rum, and more. Each item includes a detailed description, image, and modal view for an enhanced user experience.

  • Interactive UI: Smooth touch interactions with modals for detailed views.
  • Category Organization: Sectioned list for easy navigation of drink categories.
  • Cross-Platform: Built with React Native for compatibility on both iOS and Android.
