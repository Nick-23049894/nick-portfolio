About the Project

GitHub Video

The React Native Alcohol List App is a dynamic mobile application designed to showcase a categorized collection of popular alcoholic beverages. With a sleek user interface and responsive features, the app allows users to explore different categories of drinks like Whiskey, Vodka, Rum, and more. Each item includes a detailed description, image, and modal view for an enhanced user experience.

Overall Design Style üèó

The app follows a modern and structured UI with a clean and interactive experience. - Uses a deep navy blue (#303880) background for a rich, elegant look. - Section headers are **color-coded** for easy differentiation of alcohol types. - List items are in white, providing strong contrast against the dark background. - Rounded Borders & Shadows add a soft, modern touch. - Interactive Modal displays detailed alcohol descriptions with a larger image. - Touchable Items allow smooth navigation and interaction. - Button Styling: Standard buttons with **white text and a blue background**.
